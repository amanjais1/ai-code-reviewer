# AI Code Reviewer

An AI-powered code review application built using React, Node.js, Express, and Google Gemini API.

## Features

* Paste your code
* Get AI-powered review
* Code quality suggestions
* Performance optimization tips
* Best practices recommendations
* Security and maintainability feedback

## Tech Stack

### Frontend

* React
* Vite
* Axios
* React Markdown

### Backend

* Node.js
* Express.js
* Google Gemini API
